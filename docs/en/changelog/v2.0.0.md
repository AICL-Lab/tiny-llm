# v2.0.0

Released: 2024-01-01

We're excited to announce Tiny-LLM v2.0.0, a major release with INT8 quantization support!

## Major Features

### INT8 Quantization (W8A16)

The headline feature of v2.0.0 is full support for INT8 weight quantization:

- **50% memory reduction** for model weights
- **2x faster inference** on memory-bound workloads
- **Minimal accuracy loss** with per-group quantization

```cpp
// Load quantized model
InferenceEngine engine;
engine.load("model-int8.bin");

// Generate as normal
engine.generate("Hello, world!");
```

### New Architecture

Completely rewritten core with modern C++17 design:

- `Result\<T\>` for no-exception error handling
- `KVCacheManager` for efficient generation
- Modular `TransformerLayer` architecture

### CUDA Optimizations

- Fused attention kernels
- Optimized INT8 matrix multiplication
- Improved memory coalescing

## Breaking Changes

### API Changes

The API has been redesigned for v2.0.0:

| v1.x | v2.0.0 |
|------|--------|
| `loadModel()` | `load()` |
| `generateText()` | `generate()` |
| `getError()` | `result.error()` |

### Configuration Changes

Configuration is now struct-based:

```cpp
// Old (v1.x)
engine.set("max_tokens", 100);

// New (v2.0.0)
GenerationConfig config;
config.max_tokens = 100;
engine.generate(prompt, config);
```

### Model Format

v2.0.0 uses a new model format. Convert your models:

```bash
tinyllm-convert --input model-v1.bin --output model-v2.bin
```

## New Components

### Result\<T\>

No-exception error propagation:

```cpp
Result<std::string> result = engine.generate("Hello");
if (!result.ok()) {
    std::cerr << result.error_message() << std::endl;
}
```

### KVCacheManager

Efficient key-value caching:

```cpp
KVCacheManager cache;
cache.initialize(model_config, cache_config);
auto seq_id = cache.create_sequence();
```

## Performance

### Benchmarks (LLaMA-7B on RTX 4090)

| Metric | v1.x | v2.0.0 | Improvement |
|--------|------|--------|-------------|
| Memory (INT8) | 13 GB | 6.5 GB | 50% |
| Prefill | 650 tok/s | 850 tok/s | 31% |
| Decode | 48 tok/s | 65 tok/s | 35% |

## Migration Guide

### Step 1: Update Model Files

```bash
tinyllm-convert --input old-model.bin --output new-model.bin
```

### Step 2: Update Code

```cpp
// Old
TinyLLM engine;
engine.loadModel("model.bin");
auto text = engine.generateText("Hello");

// New
InferenceEngine engine;
engine.load("model.bin");
auto result = engine.generate("Hello");
if (result.ok()) {
    std::cout << result.value() << std::endl;
}
```

### Step 3: Update Build

```bash
# v2.0.0 requires C++17
cmake -S . -B build -DCMAKE_CXX_STANDARD=17
cmake --build build -j$(nproc)
```
