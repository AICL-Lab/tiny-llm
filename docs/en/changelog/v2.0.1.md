# v2.0.1

Released: 2024-01-08

## New Features

- Added support for grouped-query attention (GQA)
- Added top-k sampling option
- Added repetition penalty parameter

## Performance Improvements

- Optimized INT8 dequantization kernel (20% faster)
- Reduced KV cache memory usage by 10%
- Improved prompt processing speed for long contexts

## Bug Fixes

- Fixed incorrect output when temperature is 0.0 (greedy decoding)
- Fixed tokenizer handling of Unicode characters
- Fixed CUDA graph capture issue on multi-GPU systems

## API Changes

### New Parameters

```cpp
GenerationConfig config;
config.top_k = 50;  // New: Top-K sampling
config.repetition_penalty = 1.1f;  // New: Repetition penalty
```

### ModelConfig Extensions

```cpp
ModelConfig config;
config.num_kv_heads = 8;  // New: For GQA support
```

## Known Issues

- Flash Attention may produce incorrect results with sequence lengths > 4096
- Windows build may require manual CUDA path configuration
