# v2.0.2

Released: 2024-01-15

## Bug Fixes

### Critical Fixes

- Fixed memory leak in KV cache when destroying sequences
- Fixed incorrect tokenization for special characters
- Fixed CUDA kernel crash on Ampere GPUs with specific sequence lengths

### Minor Fixes

- Fixed incorrect error message when model file is corrupted
- Fixed race condition in multi-threaded inference
- Fixed incorrect RMS normalization epsilon value

## Improvements

### Performance

- Reduced memory allocation overhead by 15%
- Improved token generation speed by 5-8% on RTX 40-series GPUs

### Documentation

- Added troubleshooting guide for common issues
- Updated API reference with more examples
- Added performance benchmarks for LLaMA-7B

## Compatibility

- Minimum CUDA version: 11.0 (unchanged)
- Tested on: Ubuntu 20.04/22.04, Windows 10/11
- GPU compatibility: Volta (SM70) and newer

## Migration Guide

No breaking changes from v2.0.1. Simply rebuild:

```bash
git pull origin main
cmake --build build -j$(nproc)
```
