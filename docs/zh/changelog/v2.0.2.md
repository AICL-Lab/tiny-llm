# v2.0.2

发布日期: 2024-01-15

## 错误修复

### 关键修复

- 修复了销毁序列时 KV 缓存的内存泄漏
- 修复了特殊字符的分词错误
- 修复了 Ampere GPU 上特定序列长度时的 CUDA 内核崩溃

### 次要修复

- 修复了模型文件损坏时的错误消息不正确
- 修复了多线程推理中的竞争条件
- 修复了 RMS 归一化 epsilon 值不正确

## 改进

### 性能

- 内存分配开销减少 15%
- RTX 40 系列 GPU 上 token 生成速度提升 5-8%

### 文档

- 添加了常见问题的故障排除指南
- 更新了 API 参考并添加了更多示例
- 添加了 LLaMA-7B 的性能基准测试

## 兼容性

- 最低 CUDA 版本：11.0（无变化）
- 测试平台：Ubuntu 20.04/22.04、Windows 10/11
- GPU 兼容性：Volta (SM70) 及更新版本

## 迁移指南

与 v2.0.1 没有破坏性变更。只需重新构建：

```bash
git pull origin main
cmake --build build -j$(nproc)
```
