# v2.0.1

发布日期: 2024-01-08

## 新功能

- 添加了分组查询注意力 (GQA) 支持
- 添加了 Top-K 采样选项
- 添加了重复惩罚参数

## 性能改进

- 优化 INT8 反量化内核（提速 20%）
- KV 缓存内存使用减少 10%
- 长上下文的提示处理速度提升

## 错误修复

- 修复了温度为 0.0（贪心解码）时输出不正确的问题
- 修复了分词器处理 Unicode 字符的问题
- 修复了多 GPU 系统上的 CUDA 图捕获问题

## API 变更

### 新参数

```cpp
GenerationConfig config;
config.top_k = 50;  // 新增：Top-K 采样
config.repetition_penalty = 1.1f;  // 新增：重复惩罚
```

### ModelConfig 扩展

```cpp
ModelConfig config;
config.num_kv_heads = 8;  // 新增：用于 GQA 支持
```

## 已知问题

- Flash Attention 在序列长度 > 4096 时可能产生不正确结果
- Windows 构建可能需要手动配置 CUDA 路径
